# <a name="capabilities-in-configuration-manager-technical-preview-version-1810"></a>Capacidades na versão de pré-visualização técnica do Gestor de Configuração 1810 

*Aplica-se a: Gestor de Configuração (ramo de pré-visualização técnica)*

Este artigo introduz as funcionalidades disponíveis na pré-visualização técnica para O Gestor de Configuração, versão 1810. Instale esta versão para atualizar e adicione novas funcionalidades ao seu site de pré-visualização técnica.
